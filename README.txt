ここには以下URLにあるゲーム「【バトルノベル】Dawn」に用いたファイルの一部があります。
https://unityroom.com/games/teammad_dawn
以下はそれぞれのディレクトリの説明です。

・cs
ゲームで使用されるスクリプトファイルが含まれます。言語はC#です。

・py
シナリオで使用される全ての文字を読み込むPythonファイルが含まれます。
UnityにおけるTextMeshProのFontAssetCreatorは、使用される文字に絞ったフォントを生成します。
この「使用される文字」を収集するためのPythonファイルです。
例えばpy/csv直下のcsvファイル全てから収集します。

・xlsx
シナリオをcsvファイルとして出力するためのxlsxシートです。
